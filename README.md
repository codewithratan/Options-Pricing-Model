# 📊 Options Pricing Model

A comprehensive collection of quantitative finance models for pricing financial derivatives and analyzing market risk using Python and Java. This project implements industry-standard option pricing techniques, volatility estimation methods, and hedging strategies commonly used in financial engineering and quantitative trading.

---

## 🚀 Features

- Black-Scholes Option Pricing Model
- Binomial Option Pricing Model
- Monte Carlo Option Pricing
- Implied Volatility Estimation
- Delta Hedging Simulation
- Numerical Integration Methods
- Runge-Kutta Numerical Solver
- Rough Heston Volatility Model
- Volume Participation Analysis
- Research papers and implementation notes

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Options-Pricing-Model.git
```

Navigate to the project

```bash
cd Options-Pricing-Model
```

Install dependencies

```bash
pip install numpy scipy matplotlib
```

---

## ▶️ Running the Models

Run the Black-Scholes model

```bash
python BlackScholes.py
```

Run Monte Carlo simulation

```bash
python MonteCarlo.py
```

Run Delta Hedging

```bash
python DeltaHedging.py
```

Run Implied Volatility estimation

```bash
python ImpliedVolatility.py
```
